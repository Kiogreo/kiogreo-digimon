# Digimon Naming Conventions

## Critical Rule: ALL Digimon Names Must End With "mon"

This follows the Digimon franchise tradition where every Digimon's name includes "mon" as a suffix.

---

## Why "mon"?

In the Digimon franchise:
- "mon" is short for "monster"
- It's a universal identifier that something is a Digimon
- Examples from anime/games: Agumon, Gabumon, Guilmon, Patamon, Terriermon, Veemon, Wormmon

**WITHOUT "mon" = NOT a proper Digimon name!**

---

## Name Structure Patterns

### Pattern 1: Simple Prefix + mon

**Format**: `[Characteristic/Trait] + mon`

**Examples**:
- Agu (fang/jaw) + mon = **Agumon**
- Gabu (horn/gaburi) + mon = **Gabumon**
- Guil (guild/crimson) + mon = **Guilmon**
- Pata (flap/wing) + mon = **Patamon**
- Flame (fire) + mon = **Flamemon**
- Spark (electric) + mon = **Sparkmon**
- Gear (mechanical) + mon = **Gearmon**

### Pattern 2: Compound + mon

**Format**: `[Word1] + [Word2] + mon`

**Examples**:
- Metal + Garuru + mon = **MetalGarurumon**
- War + Grey + mon = **WarGreymon**
- Mega + Dra + mon = **MegaDramon**
- Cyber + Dra + mon = **CyberDramon**

### Pattern 3: Thematic + mon

**Format**: `[Theme/Concept] + mon`

**Engineering Digimon**:
- Codemon, Technomon, Mechanmon, Gearmon, Enginemon

**Scientist Digimon**:
- Labmon, Researchmon, Experimentmon, Scientmon

**Arts Digimon**:
- Artmon, Paintmon, Designmon, Creativmon

**Politics Digimon**:
- Strategmon, Diplomatmon, Tactmon

**Business Digimon**:
- Execmon, Managemon, Businessmon

**Data-Analysis Digimon**:
- Datamon, Analysmon, Metricmon, Bitmon, Bytemon

**Creative-Writing Digimon**:
- Stormon, Wordmon, Writemon, Authormon

**Support Digimon**:
- Helpmon, Supportmon, Coordinmon

**Research Digimon**:
- Scholmon, Investigmon, Searchmon

---

## Fresh-Stage Naming Guidelines

Fresh-stage Digimon (newly hatched) typically have:
- **Simple names**: 2-3 syllables
- **Cute sound**: Easy to pronounce
- **Basic trait reference**: Simple characteristic

**Good Fresh Names**:
- Agumon (simple, cute, 3 syllables)
- Gabumon (simple, cute, 3 syllables)
- Flamemon (fire theme, simple)
- Sparkmon (electric theme, simple)
- Codemon (tech theme, simple)

**Avoid for Fresh**:
- Complex compounds (save for evolutions)
- Long names (keep it simple)
- Hard-to-pronounce combinations

---

## Evolution Stage Naming

As Digimon evolve, names can become more complex:

### Fresh → In-Training
- Keep name simple or slightly enhance
- Example: Flamemon → Flamemon (or Flamemon stays)

### In-Training → Rookie
- Can add descriptor or keep base
- Example: Flamemon → Agumon or Flamemon

### Rookie → Champion
- More elaborate names acceptable
- Example: Flamemon → MegaFlamemon or WarFlamemon

### Champion → Ultimate
- Most elaborate names
- Example: MegaFlamemon → UltimateFlamemon or PhoenixGreymon

## Name Transformation Rules

When hatching a Digitama, transform the name if it doesn't end with "mon":

### Transformation Examples

| Digitama Name | Transformation Logic | Final Digimon Name |
|---------------|---------------------|-------------------|
| FireDragon | Fire → Flame + mon | **Flamemon** |
| TechnoEgg | Techno + mon | **Technomon** |
| WisdomEgg | Wisdom → Sage + mon | **Sagemon** |
| SparkBuddy | Spark + mon | **Sparkmon** |
| DataWizard | Data + mon | **Datamon** |
| ArtSpirit | Art + mon | **Artmon** |
| CodeMaster | Code + mon | **Codemon** |
| EngineGear | Gear + mon | **Gearmon** |
| PaintBrush | Paint + mon | **Paintmon** |
| ResearchLab | Lab + mon | **Labmon** |

### Transformation Process

1. **Check if name ends with "mon"**
   - YES → Keep the name as-is
   - NO → Transform it

2. **Identify core theme**
   - Extract main concept (Fire, Tech, Data, etc.)

3. **Create simple name**
   - For Fresh stage: [Theme]mon
   - Keep it simple and cute

4. **Validate pronunciation**
   - Should roll off the tongue
   - Avoid awkward syllable combinations

## Special Cases

### User Provides Name WITH "mon"

If user creates Digitama named "Agumon", "Flamemon", etc.:
- **Keep it exactly as provided!**
- Do NOT transform or change it
- User knows what they want

### User Provides Name WITHOUT "mon"

If user creates Digitama named "FireDragon", "TechWizard", etc.:
- **Transform to proper Digimon name**
- Add "mon" suffix or modify appropriately
- Keep the theme/essence

### Multiple Words

If Digitama name has multiple words:
- Extract main theme: "Fire Dragon" → "Flame" or "Dra"
- Add mon: Flamemon or Dramon
- Keep it simple for Fresh stage

---

## Name Validation Checklist

Before finalizing a Digimon name, verify:

- [ ] Name ends with "mon"
- [ ] Name is pronounceable
- [ ] Name is appropriate for evolution stage (simple for Fresh)
- [ ] Name reflects core theme or trait
- [ ] Name doesn't conflict with existing Digimon
- [ ] Name follows franchise conventions
